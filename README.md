# Student-Marks-Analysis

This project presents a Power BI dashboard developed to analyze student academic performance data. The dashboard gives a visual summary of top rankers, high scorers, subject-wise marks, and overall pass and fail percentages. It is designed to help understand student achievement patterns and compare performance across subjects.

## Project Objective
The objective of this project is to transform raw student marks data into an interactive Power BI dashboard that helps evaluate academic performance, identify top-ranking students, and compare marks across different subjects.

## Key Questions
- Who are the top-ranking students based on total marks?
- Which students have achieved the highest overall scores?
- What is the pass and fail percentage of the class?
- How are marks distributed in the BDA subject?
- How do students perform across multiple subjects?
- What patterns can be identified from the marks data?

## Process
Student marks data was collected and imported into Power BI for analysis. The dataset included student names, subject-wise marks, total scores, and pass/fail status. Data cleaning was performed to ensure proper formatting and remove inconsistencies. The dashboard was then created using a combination of charts and visual elements including bar charts, donut charts, pie charts, and line charts to highlight important academic insights.

## Dashboard Features
- Top rankers visualization
- Higher rank score comparison
- Overall pass and fail analysis
- BDA marks distribution
- Subject-wise total marks comparison
- Interactive student performance dashboard

## Insights
- The highest total score displayed in the dashboard is approximately **164 marks**.
- The top rankers chart highlights the leading students based on overall marks.
- The higher-rank comparison visual helps identify how top performers contribute to the overall ranking distribution.
- Around **75%** of students have passed, while about **25%** have failed.
- The BDA subject marks show variation in student performance, with only a few students reaching the highest score range.
- Subject-wise comparison across **BDA, NS, CC, and DC** reveals differences in student strengths across subjects.
- Some students show consistently strong performance in all subjects, while others perform better in selected subjects only.

## Conclusion
This dashboard provides a meaningful overview of student marks and academic outcomes. It helps identify top performers, understand pass and fail distribution, and compare subject-wise scores in an easy-to-read format. The project demonstrates how Power BI can be used effectively for academic performance analysis and educational decision-making.
